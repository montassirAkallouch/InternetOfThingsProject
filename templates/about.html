DESCRIPTION


Our project, focuses on temperature and humidity measurement using DHT11 sensors connected to an Arduino.

The collected data is seamlessly transmitted to a Django application hosted on PythonAnywhere, facilitating remote monitoring.

This intelligent system incorporates a sophisticated alerting mechanism; if the temperature or humidity values surpass predefined thresholds, the Django application triggers alerts through multiple channels, including SMS, WhatsApp, Telegram, and email.

The hardware components, such as the Arduino board and DHT11 sensors, work in tandem with the software architecture, creating a reliable and efficient system.

By hosting the Django application on PythonAnywhere, the project achieves accessibility and real-time monitoring capabilities.

The comprehensive solution not only provides valuable environmental data but also ensures prompt notification and response in case of critical deviations from the specified conditions.
